> 🎨 专为小众兴趣创作者与爱好者打造的综合性平台，连接创作、交流与学习的兴趣生态圈


## 一、项目背景与介绍
### 1. 诞生契机
当前小众兴趣领域存在两大核心痛点：  
- 创作者缺乏专属展示渠道，优质内容难以触达同好；  
- 爱好者难以精准找到兴趣内容，缺乏高效交流场景。  

基于此，我们打造了 **NicheCraft Hub**，旨在解决行业痛点，为小众兴趣群体提供「展示-交流-成长」的一站式解决方案。

### 2. 产品定位
NicheCraft Hub 是专注于 **小众兴趣领域** 的网页平台，核心价值包括：  
- 为创作者提供「才华展示舞台」，支持多样化兴趣作品（如摄影、数字艺术等）的上传与管理；  
- 为爱好者搭建「同好交流家园」，通过评论互动、关键词搜索连接兴趣社群；  
- 为所有用户打造「学习成长宝库」，助力在兴趣领域持续探索与提升。


## 二、核心功能模块
| 功能分类       | 具体说明                                                                 |
|----------------|--------------------------------------------------------------------------|

## 六、使用流程说明
1. **账号注册/登录**：打开平台首页，新用户点击「注册」填写信息，老用户直接「登录」；  
2. **浏览/搜索作品**：登录后进入首页，可直接浏览推荐作品，或通过顶部搜索框输入关键词查找兴趣内容；  
3. **发布作品**：点击导航栏「创作」→「发布新作品」，上传图片并填写作品描述，确认后完成发布；  
4. **管理作品**：进入「个人中心」→「我的作品」，可查看所有已发布作品，或删除不需要的内容；  
5. **互动交流**：点击任意作品进入详情页，在评论区输入观点并提交，与作者或其他爱好者交流。

1. **Fork 仓库**：点击 GitHub 仓库页面右上角「Fork」，创建个人分支；
